My Work

Stable Marriage Problem

Developed an efficient C++ algorithm to solve the Stable Marriage Problem. This project demonstrates my proficiency in algorithm design and problem-solving by providing a stable matching between two sets based on given preferences, ensuring that the solution is both optimal and stable.

Hanoi Towers

Created a C++ program to solve the Tower of Hanoi problem, which is a classic algorithmic puzzle. The program effectively calculates the minimal number of moves required to transfer a set of disks from one peg to another, adhering to the problem's constraints. This showcases my understanding of recursion and algorithm optimization.

Shortest Path Problem

Implemented a C++ program to find the shortest path in a grid, using dynamic programming. The project calculates the minimum path cost from the leftmost column to the rightmost column in a grid, demonstrating skills in cost optimization and matrix manipulation.
